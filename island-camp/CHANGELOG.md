# 変更履歴 — 島キャンプ思い出マップ

形式は [Keep a Changelog](https://keepachangelog.com/ja/) に準拠。
バージョニングは [Semantic Versioning](https://semver.org/lang/ja/)（運用ルールは docs/DESIGN.md §12）。

## [2.3.0] - 2026-07-19
### 追加
- **観光マップ風モード**: 詳細地図に「🎨 観光マップ」ボタンを追加。地理院タイル
  （地図／航空写真）に加え、観光協会パンフ調のデフォルメSVG地図（海岸線は
  OSM実データ、スポットはカテゴリ色バッジ＋ラベル常時表示、山・港マーカー、
  タイトルリボン）へ切替可能。ズーム／パン／スポットタップ（詳細・編集）／
  地図タップでのスポット位置指定にも対応。選択は端末に記憶
- **観光協会の公式マップリンク**: 24島分の観光協会・公式観光サイトの
  マップ/パンフレットページへのリンクボタン（🌐）を詳細地図に設置
- **📸 アルバム（LINE風・端末内のみ）**: アルバム作成→カード一覧→グリッド→
  タップで拡大表示。写真に加えて動画もアップロード可（サムネ自動生成・▶バッジ・
  タップで再生）。複数選択での一括アップロード、PCからのドラッグ&ドロップ対応。
  「📅 年別」表示（アルバム＋島の写真を横断で年ごとに一覧）、年単位の一括
  ZIPダウンロード・年をまたぐ選択ダウンロード・選択削除
- **📁 ドキュメント（端末内のみ）**: PDF・Word・Excel・PowerPoint等を貼って
  一覧・閲覧・ダウンロードできるページ。種類別アイコン、D&D対応
- 島の写真タブの写真をタップで拡大表示（前後送り・スワイプ対応）
- バックアップ（JSON）にアルバム・動画・ドキュメントを含めるよう拡張
### 変更
- **詳細パネルのセクションを折りたたみ式に**: スポット／写真／旅日記／お店／
  ナレッジ／Tipsを「▸ 開く」ボタンで展開する方式に変更（件数バッジつき）。
  下までスクロールしなくても全セクションに届く
- **携帯縦ではパネルを「下から重なるシート」に**: 上部角丸＋つまみ表示、
  背後の島が見える薄暗幕つき。薄暗幕タップ／つまみを下スワイプで閉じる
- IndexedDB を3ストア化（photos / media / files。DBバージョン2へ）
### 削除
- v1系の未使用ファイル islemap.js / miyake.js を削除（観光マップ風は
  kankomap.js として v2 用に再実装。検証済みの山・港座標と観光協会リンクを継承）

## [2.0.0] - 2026-07-17（commit `fa2d56f`）
### 追加
- **島別詳細マップ**: 島選択後に国土地理院タイル（淡色／航空写真切替）の
  Leafletマップを表示。全画面表示対応
- **スポット機能**: 9島×178の有名スポットをシード収録
  （キャンプ場29箇所を最優先網羅。全座標をOSM・Wikipedia・NAVITIME等で検証済み）。
  Webリンク・メモ付きで追加／編集／削除、地図タップで位置指定、カテゴリ絞り込み
- **5人共有同期**: 非公開GitHub Gistバックエンド。共有コード（`ic1.…`）方式、
  エントリID単位のLWWマージ、削除はtombstone、更新者名を表示。
  アプリ内から共有グループ作成・共有解除が可能
- お店にもWebリンク欄を追加
### 変更
- 詳細地図を手描きSVG（実寸の1.5〜2倍のデフォルメ）から実地図タイルへ刷新
  （全国マップのデフォルメシェイプは視認性のため継続）
- Service Worker: 同一オリジンをネットワーク優先に変更（更新が確実に届く）。
  エラーレスポンスをキャッシュしない。オフラインfallbackはページ遷移のみ。
  GitHub API・地図タイルはSW対象外
- localStorage を v2 形式へ（全エントリにID・更新者・更新時刻。v1から自動移行）
### 修正（マルチエージェントレビュー 42指摘より）
- 背景同期後に古い島参照へ書き込み、追加データが消失するバグ
- 共有データ・インポートJSON経由のID属性XSS（トークン窃取まで可能だった）
- 緯度・経度の空欄が座標0として保存される問題
- 共有グループから抜けられない問題
- シード再注入により削除済みシードエントリが復活する問題
- 破損した保存データを起動時にシードで上書きしてしまう問題 ほか
### 備考
- 2026-06-28〜07-11 に別系統で開発されたUI（三宅島スペシャル、パンフレット級マップ、
  タブUI等）は 07-17 のマージで本バージョンに統合され、置き換えられた。
  残存ファイル（geo.js / islemap.js / miyake.js / design/）は未参照のレガシー
