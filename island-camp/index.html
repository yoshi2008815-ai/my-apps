データの管理

追加・編集した内容はこの端末に保存され、共有設定をすると5人で同期されます。写真は端末内のみ（共有されません）。

👥 5人で共有（同期設定）

共有コードを設定すると、スポットやメモの追加・編集がメンバー全員で同期されます。 名前を設定すると更新に「誰が書いたか」が表示されます。写真は端末内のみ（共有されません）。

はじめて共有グループを作る（代表者向け）
  1. GitHub にログイン → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token
  2. スコープは gist だけにチェックして作成
  3. できたトークンを下に貼り、「共有グループを作成」を押す
  4. 発行された共有コードを LINE などでメンバー5人に送る

※データは代表者の GitHub アカウントの非公開 Gist に保存されます。コードは各自の端末にのみ保存されます。
